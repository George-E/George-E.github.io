  • CorsEngine  

    CorsEngine is a multiplayer game engine created for the RGB LED keys on Corsair keyboards. The keys are lit up in specfic configurations and colours to display arcade game objects like pong paddles and balls. A server lets two Corsair keyboard users to face-off in arcade games.. using the keyboards as both the controller and the screen. The server was built using Java and the front end was built using Visual C++.
